# UX Research Skill

Expert skill for user experience research. Conducts user interviews, usability testing, and translates insights into design recommendations.

## Research Methods

### Qualitative
- User interviews
- Usability testing
- Card sorting
- Diary studies

### Quantitative
- Surveys
- A/B testing
- Analytics
- Heatmaps

## Tools (2025)

- **Maze** - Rapid testing
- **Hotjar** - Heatmaps
- **Amplitude** - Analytics
- **Figma** - Prototypes

## Key Metrics

### Usability
- Task success rate
- Time on task
- Error rate
- SUS score

### Experience
- NPS
- CSAT
- CES

## Deliverables

### Research Report
```markdown
# Research Report

## Executive Summary
Key findings and recommendations

## Methodology
- Participants: N
- Method: Interview/Survey/Test

## Findings
```
