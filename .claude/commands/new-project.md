# Nuovo Progetto

Comando per inizializzare un nuovo progetto web.

## Uso

```
/new-project [tipo]
```

## Tipi Disponibili

### landing-page
Landing page moderna con animazioni
- Next.js 15 + TypeScript
- Tailwind CSS 4
- shadcn/ui + Aceternity UI
- SEO ottimizzato

### saas-app
Applicazione SaaS completa
- Next.js 15 + TypeScript
- Better Auth (Google, GitHub)
- Supabase + Drizzle ORM
- Stripe payments
- Dashboard utente

### portfolio
Portfolio personale
- Next.js 15 + TypeScript
- Tailwind CSS 4
- MDX per blog
- Animazioni

### e-commerce
E-commerce base
- Next.js 15 + TypeScript
- Supabase database
- Stripe checkout
- Carrello e ordini

## Workflow Automatico

1. **Setup Progetto**
   - Crea struttura cartelle
   - Inizializza Next.js
   - Configura TypeScript
   - Setup Tailwind CSS

2. **Dipendenze**
   - Installa dipendenze base
   - Configura shadcn/ui
   - Setup linting

3. **Struttura**
   - Layout principale
   - Componenti base
   - Pagine iniziali

4. **Configurazione**
   - Environment variables
   - Database (se richiesto)
   - Auth (se richiesto)

## Checklist Post-Setup

- [ ] Variabili ambiente configurate
- [ ] Git inizializzato
- [ ] Dev server funzionante
- [ ] Build senza errori
